# API Endpoints

## HTML API

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

###Presigners
- `GET api/presign`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Tracks

- `GET /api/tracks`
- `POST /api/tracks`
- `GET /api/tracks/:id`
- `PATCH /api/tracks/:id`
- `DELETE /api/tracks/:id`

### Playlists

- `GET /api/playlist`
- `POST /api/playlist`
- `GET /api/playlist/:id`
- `PATCH /api/playlist/:id`
- `DELETE /api/playlist/:id`
- `GET /api/playlist/:id/tracks`
